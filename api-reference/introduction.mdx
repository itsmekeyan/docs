## Welcome

This documentation describes the Kuralynx API for sending calls via the `/v1/send-call` endpoint.

- **Base URL:** `https://api.kuralynx.com`
- **Authentication:** All requests require an API key in the `Authorization` header, generated from [app.kuralynx.com](https://app.kuralynx.com).

## Endpoint

### POST `/v1/send-call`

**Request Body:**
```json
{
  "agent_id": "string", // The unique identifier for the agent
  "call_to": "string"    // The phone number or recipient to call
}
```

**Headers:**
```
Authorization: <api_key generated from app.kuralynx.com>
```

**Response:**
```json
{
  "status": "success",
  "call_id": "string",
  "message": "Call successfully queued"
}
```
